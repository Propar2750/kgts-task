Create a simple notes app that allows users to create, delete, and update notes. Store all data in the browser’s local storage.

### **(Bonus) Notes App with Auth \+ Database (20 points)**

### **Task 2: The Make Cuts Game (50 points)**

Develop an interactive, web-based two-player game based on the following paper-cutting logic. You will need to build the UI, the game state manager, and the win/loss detection. 

For complete rules, refer to : [Make Cuts](https://drive.google.com/file/d/1R5rOlqp3BHEeSlHl7C3crAfoOnY0qLoc/view?usp=drive_link)

### **Task 3: The Order and Chaos AI Engine (70 points)**

For this task, develop a 6x6 grid-based game of "Order and Chaos" featuring an interactive UI and a single-player mode against a custom AI opponent

For complete rules, refer to : [Order and Chaos](https://drive.google.com/file/d/1z5weyH2J5xoKLsTMjW3HKzcizmj1PkUP/view)

**Your Task:**

1. Build the frontend UI and game logic for a playable Order and Chaos board.  
2. Implement an AI Bot that the human player can compete against.  
3. Because the game is asymmetric (the goals of Order and Chaos are completely different), your bot must use an algorithm (like Minimax) combined with a Heuristic Evaluation Function to score the board state.  
4. Evaluation: Points will be awarded based on your UI design, game state management, and the intelligence/difficulty of your AI bot.

**(Bonus) Optimized AI (30 points)**

The 6x6 board has a massive branching factor. Implement Alpha-Beta Pruning to optimize your bot's search tree, allowing it to calculate moves faster without crashing the browser.
